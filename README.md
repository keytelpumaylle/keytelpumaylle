### 🛒 Dizzgo — E-commerce platform
> Next.js · Prisma · PostgreSQL · React Native · Astro · Docker · Cloudflare
 
Plataforma de comercio electrónico de licores: tienda web, backend REST, app móvil y sitio de contenido bajo un solo dominio. Arquitectura multi-app con routing via Cloudflare Workers. Motor de recomendaciones personalizado (TF-IDF + Cosine Similarity) implementado en TypeScript sin dependencias externas. Push notifications con FCM + Firebase Admin en el flujo de pedidos. Deploy en VPS propio con Docker Compose.
 
`Next.js 15` `Prisma 7` `PostgreSQL` `React Native (Expo SDK 53)` `Astro` `Docker Compose` `Cloudflare` `FCM`
